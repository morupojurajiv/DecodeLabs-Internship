#  Project 1: AI Iris Flower Classification System

## Description

This project uses Machine Learning and the K-Nearest Neighbors (KNN) algorithm to classify Iris flowers into different species based on flower measurements.

## Features

* Dataset Loading
* Machine Learning Model Training
* Accuracy Evaluation
* Interactive Flower Prediction
* Professional AI Report

## Technologies Used

* Python
* Scikit-Learn
* Pandas
* VS Code

## How to Run

Install required libraries:

```bash
pip install scikit-learn pandas
```

Run the project:

```bash
python iris_classifier.py
```
