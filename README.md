# Iris Flower Classification using KNN

## Description
This project uses the Iris dataset and the K-Nearest Neighbors (KNN) algorithm to classify iris flowers into different species.

## Technologies Used
- Python
- Scikit-Learn
- Pandas
- VS Code

## How to Run

1. Install required libraries:
pip install scikit-learn pandas

2. Run the project:
python iris_classifier.py

## Output
- Accuracy Score
- Confusion Matrix
- F1 Score
